# Limkokwing Library Simulation (OOP Assignment)

## 📌 Description
This project is a simple simulation of a library system built using Python.  
It demonstrates how users can borrow and return books using Object-Oriented Programming (OOP) and asynchronous programming.

## 🎯 Features
- Borrow books
- Return books
- Handle multiple users at the same time using async
- Display results in the terminal

## 🛠 Technologies Used
- Python
- asyncio (for asynchronous programming)
- OOP (Classes and Objects)
